# Chrome-LiveSteam
Screenshare to website from any chrome browser

### Feature list

- [x] Auto update resolution.
- [x] Sync compression settings to every client.
- [x] Auto load on LiveStream page.
- [ ] Admin panel.
- [ ] Multiple screenshare sessions.
